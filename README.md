## Instalação

## 🏁 Para rodar o projeto:

Clone este repositório em sua máquina:

```bash
$ git clone https://github.com/future4code/Carver-labe-food3.git
```

cd `Carver-labe-food3` e rode:

```bash
npm install
```

para iniciar:

```bash
npm run start
```
